# CAGED-Geometrie: Warum fünf Formen das Griffbrett beherrschen

> **Fachbereich Gitarrenstudien** | Stufe: Mittelstufe | Dauer: 45 Minuten

## Lernziele
- Verstehen, warum genau 5 offene Akkordformen das gesamte Griffbrett abdecken
- Das CAGED-System aus der Beobachtung der Intervallstruktur offener Akkorde herleiten
- Nachweisen, dass CAGED eine mathematische Konsequenz der Standardstimmung ist, kein willkürliches System
- Verschiebbare Barre-Akkordformen anwenden, um jeden Dur-Akkord über den gesamten Hals zu navigieren

---

## 1. Die fünf offenen Formen

Jeder Gitarrist lernt diese fünf offenen Dur-Akkorde früh. Jeder hat einen eigenen geometrischen Fingerabdruck:

### C-Form
```
x 3 2 0 1 0    Saiten: 5-4-3-2-1
  G 3 5 G 3    Intervalle: Grundton-gr.3-r.5-Grundton-gr.3
```

### A-Form
```
x 0 2 2 2 0    Saiten: 5-4-3-2-1
  G 5 G 3 5    Intervalle: Grundton-r.5-Grundton-gr.3-r.5
```

### G-Form
```
3 2 0 0 0 3    Saiten: 6-5-4-3-2-1
G 3 5 G 3 G    Intervalle: Grundton-gr.3-r.5-Grundton-gr.3-Grundton
```

### E-Form
```
0 2 2 1 0 0    Saiten: 6-5-4-3-2-1
G 5 G 3 5 G    Intervalle: Grundton-r.5-Grundton-gr.3-r.5-Grundton
```

### D-Form
```
x x 0 2 3 2    Saiten: 4-3-2-1
    G 5 G 3    Intervalle: Grundton-r.5-Grundton-gr.3
```

**Zentrale Beobachtung:** Jede Form enthält nur drei Tonklassen: Grundton, große Terz und reine Quinte. Die Unterschiede liegen in der *Stimmführung* — in welcher Oktave jeder Ton erscheint und welche Saiten ihn tragen.

---

## 2. Der Verschiebetest

Hier ist die zentrale Erkenntnis: Wenn man die *Fingergeometrie* eines offenen Akkords beibehält, aber den Hals hinaufschiebt, bleiben die Intervallverhältnisse erhalten. Der Barre-Finger ersetzt den Sattel.

**E-Form bei Bund 0:** E-Dur (offen)
**E-Form bei Bund 1:** F-Dur (F-Barre-Akkord)
**E-Form bei Bund 3:** G-Dur
**E-Form bei Bund 5:** A-Dur

Dies funktioniert, weil das Intervall zwischen jedem Paar benachbarter Saiten durch die Stimmung festgelegt ist. Das Verschieben aller Finger um die gleiche Anzahl von Bünden transponiert jeden Ton um das gleiche Intervall.

Das ist kein pädagogischer Trick — es ist ein *geometrischer Lehrsatz* über das Griffbrett.

---

## 3. Warum genau fünf Formen?

Die Standardstimmung (E-A-D-G-H-E) erzeugt ein bestimmtes Intervallmuster zwischen den Saiten:

```
Saite:     6    5    4    3    2    1
Note:      E    A    D    G    H    E
Intervall:   r4   r4   r4   gr3  r4
```

Das Muster Quarte-Quarte-Quarte-Terz-Quarte erzeugt genau **fünf verschiedene Bereiche**, in denen offene Akkordformen gebildet werden können. Jede Form umfasst einen anderen Bereich von Bünden und Saiten.

Die fünf Formen greifen wie Puzzleteile ineinander:

```
Bund:  0    3    5    7    8    10   12
       |--E--|--D--|--C--|--A--|--G--|--E--|
       (Formen dargestellt für die Tonart E-Dur)
```

Die Bewegung durch die C-A-G-E-D-Reihenfolge führt den Hals hinauf, wobei der Grundton jeder Form sich mit dem höchsten Ton der nächsten Form verbindet. Dies ist die **CAGED-Sequenz** — ein zyklischer Pfad durch alle fünf Stimmführungsbereiche.

---

## 4. Ist CAGED die einzige Reihenfolge?

Die fünf Formen bilden einen **Zyklus**, sodass jeder Startpunkt eine gültige Sequenz ergibt:

- **CAGED** (am gebräuchlichsten)
- **AGEDC** (von A ausgehend)
- **GEDCA** (von G ausgehend)
- **EDCAG** (von E ausgehend)
- **DCAGE** (von D ausgehend)

Der Name „CAGED" ist eine Eselsbrücke. Der zugrundeliegende 5-Formen-Zyklus ist strukturell bestimmt. Alle Reihenfolgen beschreiben die gleiche Griffbrettaufteilung.

---

## 5. Abhängigkeit von der Stimmung

CAGED ist spezifisch für die Standardstimmung. Ändert man die Stimmung, ändern sich die Formen:

- **Reine Quartenstimmung** (E-A-D-G-C-F): Die große-Terz-Unregelmäßigkeit zwischen den Saiten 3-2 verschwindet. Akkordformen werden einheitlicher, aber anders als bei CAGED.
- **Offene G-Stimmung** (D-G-D-G-H-D): Die Leersaiten bilden bereits einen Akkord, was ein völlig anderes Formsystem erzeugt.
- **Drop-D** (D-A-D-G-H-E): Nur die tiefste Saite ändert sich, wodurch die meisten CAGED-Formen erhalten bleiben, aber die E- und G-Formen auf den Basssaiten verändert werden.

Dies beweist, dass CAGED *aus* der Geometrie der Standardstimmung *abgeleitet* ist, nicht aufgezwungen.

---

## Übung

### Übung 1: Formidentifikation
Spielen Sie einen C-Dur-Akkord in allen fünf CAGED-Formen. Beginnen Sie mit der offenen C-Form, dann finden Sie die A-Form (Bund 3), G-Form (Bund 5), E-Form (Bund 8) und D-Form (Bund 10).

### Übung 2: Formverbindung
Wählen Sie zwei benachbarte CAGED-Formen. Finden Sie die Töne, die sie auf den gleichen Saiten teilen. Diese gemeinsamen Töne sind Ihre „Drehpunkte" für den Wechsel zwischen Formen.

### Übung 3: Eine Saite, fünf Formen
Spielen Sie auf der 2. Saite allein den Ton C in jeder der fünf CAGED-Positionen. Beachten Sie, wie jede Form das C auf einen anderen Bund dieser Saite legt.

---

## Wichtigste Erkenntnisse
- Das CAGED-System ist **entdeckt, nicht erfunden** — es ist eine strukturelle Konsequenz der Geometrie der Standardstimmung
- Genau **5 Formen** decken das Griffbrett ab, bedingt durch das Stimmungsmuster Quarte-Quarte-Quarte-Terz-Quarte
- Jede Form behält ihre Intervallstruktur bei, wenn sie gebarrt und verschoben wird — das ist ein geometrischer Lehrsatz, keine pädagogische Vereinfachung
- Der Name „CAGED" ist eine von 5 gleichwertigen zyklischen Reihenfolgen
- Verschiedene Stimmungen erzeugen verschiedene Formsysteme, was bestätigt, dass CAGED stimmungsbedingt ist
