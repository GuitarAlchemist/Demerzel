# Department of Visual Computing — Course Catalog

> **Streeling University** | Head: architect | Grammar: sci-visual-computing.ebnf

## Mission

Teach AI to master the full visual rendering spectrum: 2D graphics, 3D engines, WebGPU, game engines, data visualization, animation, spatial computing, 3D modeling, and accessible design.

## Curriculum

### Foundations
- SVG fundamentals: paths, transforms, viewBox, responsive scaling
- Canvas 2D API: drawing, compositing, double-buffering, dirty rects
- CSS layout mastery: Flexbox, Grid, container queries, logical properties

### 3D Rendering
- Three.js scene graph: cameras, lights, materials, geometries, loaders
- Babylon.js PBR materials and shadow generation
- WebGPU from scratch: device, pipeline, shaders, bind groups, compute
- WGSL shader language: vertex, fragment, compute entry points

### Data Visualization
- D3.js data joins: enter-update-exit, scales, axes, transitions

### Reactive Frameworks
- React component architecture: hooks, context, suspense, server components
- Svelte reactivity: runes, stores, actions, transitions

### Animation
- GSAP animation: tweens, timelines, ScrollTrigger, MorphSVG
- Framer Motion: layout animations, AnimatePresence, gestures
- Spring physics: damping ratio, natural frequency, critically damped systems

### Spatial Computing
- WebXR session management: VR, AR, hit testing, hand tracking

### Game Engines
- Godot GDScript: scene tree, signals, exports, TweenNode

### 3D Modeling
- Blender Python (bpy): mesh creation, modifiers, materials, rendering

### Post-Processing & Performance
- Post-processing chains: bloom, SSAO, DoF, color grading, LUTs
- Render-critic loop: screenshot, score, adjust, converge
- GPU performance profiling: draw calls, fill rate, overdraw, instancing

### Accessibility
- WCAG accessibility: contrast ratios, ARIA roles, screen reader support, reduced motion

## Course Production

Courses are produced by Seldon research cycles. When a research question achieves T (confirmed) status, the finding is packaged as a course module in this directory.
