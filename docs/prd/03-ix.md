# PRD: ix -- Rust ML Forge

**Version:** 1.0 | **Last Updated:** 2026-04-03 | **Status:** Active

---

## Executive Summary

ix is a Rust workspace of 32 crates implementing foundational ML/math algorithms from scratch -- no external ML frameworks. It spans linear algebra, supervised/unsupervised learning, deep learning, reinforcement learning, game theory, signal processing, chaos theory, topology, and GPU compute via WGPU. Exposed as 37 MCP tools via JSON-RPC and as a CLI binary, ix serves as the ML backbone of the GuitarAlchemist ecosystem.

## Problem Statement

The ecosystem needs composable, high-performance ML primitives that agents can invoke as tools. Off-the-shelf Python ML libraries are not suitable for an agent-native workflow requiring MCP tool integration, Rust safety guarantees, and governance-aware execution. ix fills this gap with pure Rust implementations that agents call through the MCP federation.

## Goals & Success Metrics

### P0 (Must-Have)
- **Build clean**: `cargo build --workspace` + `cargo clippy -- -D warnings` pass on stable + nightly
- **Test coverage**: Unit tests for every public function, property tests for math invariants
- **MCP server operational**: 37 tools accessible via JSON-RPC stdio

### P1 (Should-Have)
- **GPU compute**: WGPU-based cosine similarity, matrix multiply, BSP kNN functional on Vulkan/DX12
- **Benchmarks**: Criterion benchmarks for performance-critical paths
- **Federation**: MCP federation with tars and ga servers

### P2 (Nice-to-Have)
- **Pipeline executor**: DAG-based skill orchestration with memoized data flow
- **Probabilistic data structures**: Bloom, Count-Min, HyperLogLog, Cuckoo operational
- **Demo app**: egui desktop with 22+ interactive algorithm tabs

## Architecture

```
Crate Dependency Flow:
  ix-math ──► ix-supervised ──► ix-ensemble
     │              │
     ├──► ix-nn ◄───┘
     ├──► ix-unsupervised
     ├──► ix-signal, ix-chaos
     └──► ix-gpu (WGPU f32)

Integration Layer:
  ix-agent (MCP server, 37 tools) ◄── ix-skill (CLI)
       ▲
  ix-governance (Demerzel submodule)

Runtime: Rust 1.80+, f64 CPU / f32 GPU, ndarray + wgpu 28
CI: GitHub Actions on stable + nightly, Linux + Windows
```
