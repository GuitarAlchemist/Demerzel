# TARS v1 Chat Extraction Sources

This directory contains notes extracted from TARS v1 exploratory conversations.

## Important

These files are **extraction sources**, not governance artifacts.

- Do NOT copy raw chat transcripts into `personas/` or `constitutions/`
- Instead, distill insights into canonical formats using the schemas in `schemas/`
- Document provenance: which chat, what date, what was extracted

## Extraction Process

1. Read source chat
2. Identify behavioral patterns, principles, or constraints
3. Transform into canonical YAML/MD format
4. Validate against schema
5. Add provenance note
6. Place in appropriate directory (`personas/`, `constitutions/`, `policies/`)
