## Demerzel Commands (Governance Coordinator)

| Command | Purpose |
|---------|---------|
| `/demerzel` | Dispatcher — show all subcommands |
| `/demerzel audit` | Governance audit (Level 1: schema, Level 2: cross-ref, Level 3: full) |
| `/demerzel recon [repo]` | Three-tier reconnaissance (self-check → environment → situational) |
| `/demerzel directive [type] [repo]` | Issue governance directive via Galactic Protocol |
| `/demerzel promote [stage]` | Propose governance promotion (pattern→policy or policy→constitutional) |
| `/demerzel evolve [view]` | Governance evolution insights (summary, promotions, waste, full) |
| `/demerzel teach` | Invoke Seldon for knowledge transfer (delegates to /seldon teach) |
| `/demerzel loop [start\|status\|halt]` | Governed Ralph Loops with graduated oversight |
| `/demerzel patterns [pattern]` | Agentic patterns catalog (7 patterns mapped to governance) |

| `/demerzel report` | Post governance report + Seldon digest to Discussions |

| `/demerzel skills` | This command — list all available skills |
